"API-консолька" консоль для отправки запросов к Sendsay API. Проект на React с Redux.
https://sendsay.ru/api/api.html - описание апи;

Реализовано:

форма авторизации, с валидацией логина и пароля;
сохранение авторизации в течение суток;
сохранение истории запросов;
форматирование JSON запроса;
валидация и отображение ошибки запроса;
изменение и сохраниение ширины полей ввода;

Запуск проекта:

Установить зависимости: npm i
Запусить проект: npm start или
Собрать build командой npm run build
Запустить командой serve -s build

Структура проекта:

src/index.js - стартовый файл проекта;
src/store - файлы формирующие хранилище redux;
src/container - модули подключенные к хранилищу redux, а также компоненты уникальные для данного модуля;
src/components - переиспользуемые компоненты;
src/api - модуль реализующий подулючение к api сервера;
src/assets - дополнительные файлы проекта;

В проекте подключены дополнительные библиотеки:

"redux" - организация работы с общим хранилищем данных, однонаправленное изменение состояния хранилища;
"redux-thunk" - позволяет использовать функции при отправке событий redux;
"react-router-dom" - маршрутизация;
"lodash" - методы для работы с массивами, объектами;
"react-onclickoutside" HOC - для обработки клика вне компонента
